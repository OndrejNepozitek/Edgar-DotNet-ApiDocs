Class GeneratorPlanner<TLayout, TNode>

Class that builds a tree of nodes where each node represent a layout that can be used as an initial layout to generate more layouts until a full layout is generated.

Remarks

Each node in the tree has the number of added chains equal to the number of the row on which it is located.

Inheriting classes can shape the tree by specifying in each step which node should be extended.
